What is Android Developemnt

Android software development is the process by which applications are created for devices running the Android operating system.Google states that "Android apps can be written using Kotlin, Java, and C++ languages" using the Android software development kit (SDK), Here we will see the path to became an Android Developer.

Ingredients

  • Any programming language like Java,Kotlin etc.
  • OOPs Concept
  • Android Studio
  • Android Apis
  • Fire Base
  • Unit testing
  • UI/UX Desining
  • Database
